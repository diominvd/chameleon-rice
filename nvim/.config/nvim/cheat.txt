󰘚 ШПАРГАЛКА VIM 󰘚
 ──────────────────────────────
  [ РЕЖИМЫ ]
  i   - Вставка
  Esc - Режим команд (Normal)
  v   - Выделение (Visual)
  V   - Выделение строк
  :   - Ввод команды

  [ ПЕРЕМЕЩЕНИЕ ]
  h j k l - ← ↓ ↑ →
  w / b   - на слово вперёд / назад
  0 / $   - в начало / конец строки
  gg / G  - в начало / конец файла
  %       - к парной скобке

  [ РЕДАКТИРОВАНИЕ ]
  x       - удалить символ
  dw      - удалить слово
  dd      - удалить строку
  yy      - копировать строку
  p       - вставить
  u       - отмена (undo)
  Ctrl+r  - вернуть (redo)
  r       - заменить символ

  [ ПОИСК ]
  /текст  - поиск
  n / N   - след. / пред. совпадение
  :noh    - убрать подсветку поиска

  [ ОКНА И ФАЙЛЫ ]
  :w      - сохранить
  :q      - выйти
  :wq     - сохранить и выйти
  :vsp    - вертик. разделение
  Ctrl+w w- переход между окнами

  [ TMUX (Твой конфиг) ]
  C-a |   - разбить по вертикали
  C-a -   - разбить по горизонтали
  C-a h/j/k/l - навигация
  C-a z   - развернуть панель (зум)
 ──────────────────────────────
